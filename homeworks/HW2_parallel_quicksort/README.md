# HW2 — Parallel Quicksort: Experiment Design & Performance Analysis

This homework follows the SMPE course (02/10 lecture).

## Objectives
- Simulate or run the multithreaded Quicksort experiment
- Improve the experimental design
- Extract logs into a CSV file
- Analyze performance scaling
- Produce clear visualizations with Python (matplotlib + seaborn)
- Compute confidence intervals
- Write a short methodological report

## Folder Structure
scripts/ → helper scripts
data/raw/ → raw log files (empty)
data/processed/results.csv → synthetic realistic dataset
analysis/quicksort_analysis.ipynb → Python Jupyter notebook
analysis/design_experiment.md → experiment design description
report/hw2_quicksort_report.md → final written summary
